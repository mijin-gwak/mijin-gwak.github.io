Amazon Web Scraping


Tools Used Python, Jupyter Notebook
Skills Used Web scraping

This Python Jupyter Notebook project automates the tracking of a dumbbell rack's price on Amazon. Utilizing BeautifulSoup for web scraping, the script captures and logs real-time data, including the product title, current price, and date. As an added bonus, the system sends a notification email if the price drops below a set threshold.
